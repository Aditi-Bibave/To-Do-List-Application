A full-stack task management web application that allows users to create accounts,
log in securely, and manage their personal tasks with priorities and due dates.

## Tech Stack
- Frontend: HTML, CSS, JavaScript
- Backend: PHP
- Database: MySQL
- Server: WAMP

## Features
- User authentication (Signup & Login)
- Add, update, and delete tasks
- Task priority levels (High, Medium, Low)
- Task start date and due date
- Task descriptions
- User-specific task management
- Secure session handling

## How to Run Locally
1. Install WAMP Server
2. Clone this repository
3. Move the project folder to `wamp64/www`
4. Import `database/task_manager.sql` into phpMyAdmin
5. Update database credentials in `config/db.php`
6. Open browser and visit:
   http://localhost/task-manager
